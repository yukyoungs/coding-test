# 자릿수 더하기

## 문제 설명

자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를 만들어 주세요.

## 제한 사항

- N의 범위 : 100,000,000 이하의 자연수

## 입출력 예

| N | answer |
|---|--------|
| 123 | 6 |
| 987 | 24 |

## 풀이 방법론

N을 `String.valueOf()`로 문자열로 변환한 뒤, for문으로 한 글자씩 순회했다.

각 문자(`char`)를 `Character.getNumericValue()`로 정수로 변환해 누적합에 더하는 방식으로 풀었다.
