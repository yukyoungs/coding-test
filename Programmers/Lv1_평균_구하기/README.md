# 평균 구하기

## 문제 설명

정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성해보세요.

## 제한 사항

- arr은 길이 1 이상, 100 이하인 배열입니다.
- arr의 원소는 -10,000 이상 10,000 이하인 정수입니다.

## 입출력 예

| arr | return |
|-----|--------|
| [1,2,3,4] | 2.5 |
| [5,5] | 5 |

## 풀이 방법론

배열 길이를 `arr.length`로 구하고, while문으로 배열을 순회하며 모든 원소를 합산했다.

합계를 배열 길이로 나눌 때 두 값 모두 int라 소수점이 잘릴 수 있어서, `(double)`로 형변환해 나눗셈을 실수 연산으로 처리했다.
